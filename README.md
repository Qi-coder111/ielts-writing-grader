# AI-powered IELTS Writing Platform

> **An AI-powered IELTS Writing Assistant that provides comprehensive essay evaluation, inline corrections, writing guidance, and personalized learning resources.**

[Python]
[Flask]
[OpenAI]
[IELTS]

## 📖 Overview

IELTS Writing Grader is an AI-powered web application designed to support IELTS Writing Task 2 preparation.

Unlike traditional essay scoring tools that only provide a band score, this project offers a complete AI-assisted learning workflow. From detailed evaluation and sentence-level corrections to personalized writing ideas, topic-specific sample essays, and advanced vocabulary recommendations, learners receive comprehensive feedback that supports continuous improvement.

The goal is not simply to grade essays, but to help users understand why their writing can be improved and how to write more effectively in future tasks.

## ✨ Core Features

| Feature | Description |
|----------|-------------|
| 📊 Multi-dimensional Scoring | Evaluate essays using IELTS official assessment criteria. |
| ✍ Sentence-level Corrections | Detect grammar, vocabulary and sentence issues with inline AI suggestions. |
| 🔄 AI Rewrite | One-click sentence optimization while preserving original meaning. |
| 💡 Writing Ideas | Generate multiple arguments and supporting ideas for the topic. |
| 📚 Band 7+ Sample Essay | Automatically generate a high-band reference essay. |
| 📖 Advanced Vocabulary | Recommend topic-related high-band vocabulary and collocations. |
| 🧠 Sentence Patterns | Suggest advanced grammatical structures for future writing. |
